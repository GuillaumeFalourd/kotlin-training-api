### Requirements

1. Java - 11

2. Maven - 3.x.x

### Running the App**

Type the following command in your terminal to run the app:

```bash
mvn spring-boot:run
```

The app will start running at <http://localhost:8080>.

### Explore Rest APIs

The app defines following CRUD APIs.

    GET /api/users
    
    POST /api/users
    
    GET /api/users/{id}
    
    PUT /api/users/{id}
    
    DELETE /api/users/{id}

You can test them using any Rest Client.
